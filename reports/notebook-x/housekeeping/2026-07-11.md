### 1. Implement `dataQuality` Validation in `data_safety.py`
Currently, the `dataQuality` field is a contract requirement for the Data Center project. We must ensure that no notebook is pushed to GitHub with an invalid `dataQuality` status, which would break the `_index-public.json` contract.

**Action:** Update `data_safety.py` to include a schema validator for the notebook metadata.

```python
# Add to data_safety.py
ALLOWED_QUALITY = ["draft", "partial", "complete"]

def validate_notebook_metadata(metadata):
    """Ensures metadata conforms to the Data Center API contract."""
    if metadata.get("dataQuality") not in ALLOWED_QUALITY:
        raise ValueError(f"Invalid dataQuality. Must be one of {ALLOWED_QUALITY}")
    if not all(k in metadata for k in ["id", "name", "domain"]):
        raise KeyError("Missing required metadata fields for public index.")
    return True
```
