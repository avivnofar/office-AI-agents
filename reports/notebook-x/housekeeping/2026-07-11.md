### 3. Add `dataQuality` Validation Middleware
The `Data Center` worker depends on the `dataQuality` field. Currently, there is no enforcement of this field's value. If a user sets it to an invalid string, the Data Center worker might break.

**Action:** Update `data_safety.py` to include a schema validator for notebook metadata.

**Code Change (`data_safety.py`):**
```python
ALLOWED_QUALITIES = ["draft", "partial", "complete"]

def validate_notebook_metadata(metadata):
    if "dataQuality" in metadata and metadata["dataQuality"] not in ALLOWED_QUALITIES:
        raise ValueError(f"Invalid dataQuality. Must be one of {ALLOWED_QUALITIES}")
    return True
```
